# Smart Things

I made a concept application in React Native. Control smart devices within your own home.

## Installation

- npm install
- pods install
  try it and enjoy

## Development

|         | Debug                         |
| ------- | ----------------------------- |
| android | $npx react-native run-android |
| ios     | $npx react-native run-ios     |
